## Running Server Locally

1. Install Python dependencies
    ```sh
    pip3 install -r requirements.txt
    ```
1. Initiate the Sqlite3 database with flask CLI command
    ```sh
    flask init-db
1. Start the Flask server
    ```sh
    python app.py
    ```
1. Access the application from browser
    ```sh
    http://localhost:5000
    ```
